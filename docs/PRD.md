# 产品需求文档 (PRD)

## 1. 产品概述 (重构版)
本项目是一个基于人工智能的沉浸式试卷分析平台。用户上传试卷后，系统调用阿里云读光 OCR 完成切题与坐标标注，并通过硅基流动 DeepSeek V3.2 提供题目分析、启发式答疑与举一反三训练。核心交互从旧版左右分屏升级为“全屏阅读 + 悬浮浮窗”。

## 2. 目标用户
- 需要辅导作业或备考的学生。
- 希望通过举一反三来巩固知识点的学习者。

## 3. 核心功能
### 3.1 文件上传与处理 (修正)
- **支持格式**：图片 (JPG, PNG) 和 PDF。
- **PDF处理**：如果用户上传PDF文件，系统会自动提取第一页并将其转换为图片以供展示和分析。
- **智能 OCR 识别**：必须模拟调用阿里云读光教育场景 OCR 接口对上传的试卷进行切题分析，提取文本（包含 LaTeX 公式），并返回精确的边界框（Bounding Box）坐标。绝不能使用大模型视觉能力替代专业 OCR。

### 3.2 历史记录与恢复
- 首页展示历史试卷列表，支持一键继续学习。
- 历史记录持久化到 localStorage，记录字段包含 `id / name / date / questions / image(pdf)_url`。
- 恢复时同时恢复题目集与题号进度。

### 3.3 沉浸式工作区 (Workspace)
- **全屏阅读器**：
  - 采用 `react-zoom-pan-pinch` 展示原卷。
  - 题目边界框以百分比定位，跟随缩放和平移。
  - UI 不展示 OCR 纯文本，但题目 content 仍保留用于 AI 请求。
- **右侧胶囊导航**：
  - 右侧固定毛玻璃导航条，纵向题号快速跳转。
  - 选题后视图平滑居中到目标题目坐标区域。
- **底部操作栏**：
  - 固定三按钮：`📖 题目分析`、`💬 对话答疑`、`🎯 举一反三`。
  - 三个面板互斥，仅一个可打开。

### 3.4 悬浮窗交互
- **题目分析**：展示考点解析与答案思路，支持 Markdown + LaTeX。
- **对话答疑**：AI 主动开场并按苏格拉底式追问，必要时提供完整解析与答案。
- **举一反三**：
  - 后端强制返回 JSON 结构：`content / type / options / answer / analysis`。
  - 前端根据题型渲染作答组件（选择题/输入题），支持提交校验与解析反馈。
- **请求控制**：
  - 面板关闭、题目切换时立即 Abort 进行中的 AI 请求。
  - 采用骨架屏与动态等待态，降低等待焦虑。

## 4. 技术栈 (当前)
- **核心框架**：Next.js (App Router), React, TypeScript
- **UI 组件库**：shadcn/ui (配合 Tailwind CSS 和 Lucide React)
- **状态管理**：Zustand
- **Markdown & 公式渲染**：react-markdown, remark-math, rehype-katex
- **沉浸阅读与缩放**：react-zoom-pan-pinch
- **大模型 API**：硅基流动 DeepSeek V3.2
- **OCR 服务**：阿里云读光 API (模拟)

## 5. 语言与本地化
- 界面语言：中文 (简体)。
- AI 回复语言：中文 (简体)，数学公式使用标准的 LaTeX 语法。
