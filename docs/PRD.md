# 产品需求文档 (PRD)

## 1. 产品概述 (修正)
本项目是一个基于人工智能的响应式学习与试卷分析平台。用户上传试卷后，系统将调用阿里云读光 OCR 自动识别试卷中的题目和数学公式（LaTeX格式），并在界面上通过边界框标注。用户选择特定题目后，通过接入硅基流动 DeepSeek V3.2 / R1 模型，与 AI 进行苏格拉底式对话、获取知识点或生成变式题。

## 2. 目标用户
- 需要辅导作业或备考的学生。
- 希望通过举一反三来巩固知识点的学习者。

## 3. 核心功能
### 3.1 文件上传与处理 (修正)
- **支持格式**：图片 (JPG, PNG) 和 PDF。
- **PDF处理**：如果用户上传PDF文件，系统会自动提取第一页并将其转换为图片以供展示和分析。
- **智能 OCR 识别**：必须模拟调用阿里云读光教育场景 OCR 接口对上传的试卷进行切题分析，提取文本（包含 LaTeX 公式），并返回精确的边界框（Bounding Box）坐标。绝不能使用大模型视觉能力替代专业 OCR。

### 3.2 试卷工作区 (Workspace)
- **左侧面板（原卷展示）**：
  - 展示处理后的试卷图片。
  - 在图片上叠加所有识别出的题目边界框。
  - 用户点击某个边界框即可激活该题目，进入分析模式。
- **右侧面板（题目分析与交互）**：
  - **识别结果展示**：展示当前选中题目的文本内容，完美渲染LaTeX数学公式。
  - **知识点讲解**：点击按钮，AI将解析该题目背后的核心知识点（不直接给出答案）。
  - **相似题目生成**：点击按钮，AI将基于当前题目生成两道相似的变体题目及详细解答。
  - **苏格拉底式AI导师**：提供一个聊天界面，AI扮演导师角色，通过启发式的提问引导用户自己思考并得出答案，而不是直接给出答案。聊天过程中AI可能会提供几个选项供用户点击回复。

## 4. 技术栈 (严格限定)
- **核心框架**：Next.js (App Router), React, TypeScript
- **UI 组件库**：shadcn/ui (配合 Tailwind CSS 和 Lucide React)
- **状态管理**：Zustand
- **Markdown & 公式渲染**：react-markdown, remark-math, rehype-katex
- **分屏交互**：react-resizable-panels (shadcn 推荐方案)
- **大模型 API**：硅基流动 DeepSeek V3.2 API (用于日常交互) / 配合 DeepSeek R1 (用于复杂理科逻辑推导)
- **OCR 服务**：阿里云读光 API (模拟)

## 5. 语言与本地化
- 界面语言：中文 (简体)。
- AI 回复语言：中文 (简体)，数学公式使用标准的 LaTeX 语法。
